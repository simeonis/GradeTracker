# Grade Tracker
An android app to calculate your grades and GPA.  

## Installation
Clone this repository and import into **Android Studio**
```bash
git clone https://github.com/simeonis/GradeTracker.git
```
## Technologies Used

-**Kotlin** -**Android** -**XML**  

## Key Project Implementations
* Database and Room 
* Navigation Drawer
* Databinding & ViewModel
